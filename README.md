# 🏥 QueueLess India — Next-Gen Hospital Queue SaaS

QueueLess India is a high-performance, real-time hospital queue management system designed to eliminate physical waiting lines and provide patients with dynamic, live wait-time updates. 

![QueueLess India Preview](https://img.shields.io/badge/Status-Live-success?style=for-the-badge&logo=firebase)
![Tech Stack](https://img.shields.io/badge/Tech-React%20%7C%20Firebase%20%7C%20Tailwind-blue?style=for-the-badge)

## ✨ Features

- **Real-Time Token Management**: Instant token generation and status updates using Firebase Firestore.
- **Dynamic Wait-Time Calculation**: Smart algorithms calculate estimated wait times based on live queue density.
- **Smart Alerts**: Contextual notifications (e.g., "Head over now!" or "No rush") based on patient proximity to their turn.
- **Premium Medical UI**: A clean, high-contrast "Medical Blue & White" aesthetic designed for maximum clarity and accessibility.
- **Secure Authentication**: Robust Email/Password authentication system powered by Firebase Auth.
- **Responsive Dashboard**: Full-featured analytics and queue monitoring for both patients and administrators.

## 🚀 Tech Stack

- **Frontend**: React.js (Vite)
- **Styling**: Tailwind CSS (Vanilla CSS & Utility First)
- **Backend/Database**: Firebase Firestore
- **Authentication**: Firebase Auth
- **Real-time**: Firebase SDK (onSnapshot / onAuthStateChanged)

## 🛠️ Setup & Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/harshitlakhara7877/queueless-India.git
   cd queueless-India
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Firebase**:
   Ensure `src/firebase.js` contains your Firebase project configuration.

4. **Run in development mode**:
   ```bash
   npm run dev
   ```
